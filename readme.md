# Z-Axis Thermal Error Analysis and Compensation Study for a CNC Vertical Machining Center

## Overview

This project investigates thermal-induced positioning errors in the Z-axis of a Vertical Machining Center (VMC) and develops a compensation methodology based on machine operating conditions.

During prolonged machining operations, heat generated by spindle activity and repeated machine cycles can cause thermal growth in machine components. This thermal growth may result in Z-axis positioning drift, affecting machining accuracy and repeatability.

The objective of this project is to study these thermal effects, analyze their influence on machine accuracy, and develop a method to estimate and compensate for positioning errors.

---

## Problem Statement

Thermal effects are one of the major contributors to positioning inaccuracies in CNC machines.

As machine operating time increases, spindle operation and repeated machine cycles generate heat that can cause dimensional changes in machine components. These changes may lead to:

* Z-axis positioning drift
* Reduced machining accuracy
* Dimensional deviations
* Reduced repeatability

Understanding and compensating for these effects is essential for maintaining machining precision.

---

## Project Objectives

* Study thermal effects influencing the Z-axis of a CNC Vertical Machining Center.
* Analyze the relationship between machine operating conditions and positioning error.

* Compare different prediction approaches.
* Estimate compensation values to improve positioning accuracy.
* Identify the machine parameters that contribute most significantly to thermal error.

## Machine Parameters Considered

The study focuses on machine operating parameters such as:

* Machine Runtime
* Spindle Speed (RPM)
* Spindle Operating Duration
* Start/Stop Frequency
* Commanded Z-Axis Position
* Probe-Measured Position Error

Additional engineered parameters include:

* Heat Index
* Start-Stop Stress Index

---

## Methodology

### 1. Data Collection

Machine operating parameters and probe measurements are recorded periodically during machine operation.

Typical dataset structure:

| Time | Spindle RPM | Runtime | Start/Stop Count | Command Position | Error |
| ---- | ----------- | ------- | ---------------- | ---------------- | ----- |
| ...  | ...         | ...     | ...              | ...              | ...   |

---

### 2. Feature Engineering

Additional indicators are generated from machine operating data.

#### Heat Index

Represents cumulative thermal loading generated by spindle operation.

Heat Index = Spindle RPM × Runtime

#### Start-Stop Stress Index

Represents additional thermal loading caused by repeated spindle acceleration and deceleration.

Start-Stop Stress = StartStopCount × SpindleRPM

---

### 3. Model Development

Two prediction approaches are evaluated:

#### Linear Regression

Used to establish a mathematical relationship between machine operating parameters and positioning error.

#### Random Forest Regression

Used to capture more complex relationships between machine operating conditions and thermal drift.

---

### 4. Performance Evaluation

Model performance is evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* Coefficient of Determination (R²)

---
### 5. Compensation Estimation

The predicted positioning error is used to calculate a compensation value.

Corrected Position = Commanded Position − Predicted Error

This allows thermal drift to be considered before machine movement occurs.
